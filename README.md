## Plus、Pro 和 Codex 有什么区别

ChatGPT Plus、Pro 和 Codex 并不是完全相同的产品概念。

| 类型 | 适合人群 | 常见用途 |
|---|---|---|
| Plus | 日常用户 | 写作、翻译、图片、文件分析、轻度编程 |
| Pro | 高频用户 | 长时间使用、复杂任务、深度研究、高频 Codex |
| Codex | 开发者 | 阅读项目、修改代码、运行测试、排查错误 |

轻度使用一般先从 Plus 开始，高频开发、长期处理复杂任务时，再考虑 Pro。

<a id="订阅前准备"></a>

## 订阅前需要准备什么

开始前建议确认：

1. ChatGPT 账号可以正常登录；
2. 当前邮箱是自己长期使用的邮箱；
3. 已经了解 Plus 和 Pro 的区别；
4. 确认订阅的是 ChatGPT 会员，而不是 API 余额；
5. 提交前核对账号和套餐。

<a id="订阅方式"></a>

## 国内用户有哪些订阅方式

常见方式包括：

### 方式一：官方支付页面

适合有可用海外支付方式，并且熟悉账单地址和支付环境的用户。

### 方式二：苹果应用内订阅

适合长期使用 iPhone、iPad，并且 Apple ID 订阅环境稳定的用户。

### 方式三：国内自助订阅服务

适合不熟悉海外付款流程，希望使用国内支付方式完成订阅的用户。

<a id="套餐选择"></a>

## 如何选择适合自己的套餐

### 适合 Plus 的情况

- 日常问答和资料整理；
- 写文章、做图片、分析文件；
- 偶尔使用 Codex；
- 使用频率不算特别高。

### 适合 Pro 的情况

- 每天高频使用 ChatGPT；
- 经常进行深度研究；
- 长时间使用 Codex；
- 处理完整项目或多文件任务；
- Plus 使用额度经常不足。

<a id="状态确认"></a>

## 订阅后如何确认状态

完成订阅后，可以：

1. 退出账号后重新登录；
2. 打开 ChatGPT 设置；
3. 查看当前套餐状态；
4. 确认订阅账号是否正确；
5. 保存订单或付款记录。

<a id="常见问题"></a>

## 常见问题

### ChatGPT 会员和 API 是一回事吗？

不是。ChatGPT Plus 或 Pro 主要用于 ChatGPT 产品，API 通常需要单独计费。

### Plus 是否包含 Codex？

具体可用功能和额度需要以当前账号页面显示为准。不同套餐的使用额度和权限可能不同。

### 订阅后为什么没有马上显示？

可以尝试退出重新登录、更换浏览器或使用无痕模式，并确认登录邮箱是否与订阅账号一致。

<a id="问题排查"></a>

## 问题排查

遇到状态异常时，建议依次检查：

- 登录账号是否正确；
- 套餐是否选择正确；
- 页面是否需要刷新；
- 是否存在多个 ChatGPT 账号；
- 订单状态是否已经完成；
- 是否保存了订单号和相关凭证。
